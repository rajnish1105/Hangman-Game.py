# Description:
A classic interactive command-line game that challenges users to guess a hidden word from a specific category (Animals). This project showcases logic flow, state management, and user input validation in Python.

# key Technical Features:
State Visualization: Features a dynamic ASCII art progression that "draws" the hangman as the user loses lives.

Game Logic: Manages correct and incorrect guess tracking, prevents duplicate guesses, and calculates remaining attempts.

Randomization: Utilizes the random module to ensure a unique gameplay experience in every session.
